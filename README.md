## How to setup the project

### Step 1 - Clone the project
```
git clone https://github.com/hayk891/dynamic-form-designer.git
```

### Step 2 - installation of dependencies

```
cd dynamic-form-designer
npm install
```

 
### Step 3 - run project

```
Local run - ng serve
```
